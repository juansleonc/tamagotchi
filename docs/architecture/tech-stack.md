# Technology Stack

## Backend

### Ruby on Rails
- **Version**: 7.x or 8.x
- **Purpose**: API backend and GraphQL server
- **Key Gems**:
  - `graphql-ruby`: GraphQL implementation
  - `rspec-rails`: Testing framework
  - `pg`: PostgreSQL adapter
  - `jwt`: Authentication tokens
  - `rack-cors`: CORS handling

### Database
- **PostgreSQL**: Primary database
- **Version**: 12+ recommended

## Frontend

### React Native
- **Version**: 0.72+
- **Purpose**: Mobile application framework
- **Key Libraries**:
  - `@react-navigation/native`: Navigation
  - `@apollo/client`: GraphQL client
  - `react-native-testing-library`: Testing
  - `jest`: Test runner

### GraphQL
- **graphql-ruby** (Backend)
- **Apollo Client** (Frontend)
- **Purpose**: API communication layer

## Development Tools

### Code Quality
- **RuboCop**: Ruby linting and formatting
- **ESLint**: JavaScript/TypeScript linting
- **Prettier**: Code formatting

### Testing
- **RSpec**: Rails testing
- **Jest**: JavaScript/React Native testing
- **React Native Testing Library**: Component testing

### Documentation
- **Context7**: Auto-generated documentation
- **Markdown**: Documentation format

## Version Control

- **Git**: Source control
- **GitHub**: Repository hosting (or equivalent)

## Deployment (Future)

- **Heroku/Railway**: Backend hosting
- **Expo/EAS**: Mobile app distribution
- **App Store/Play Store**: App distribution
